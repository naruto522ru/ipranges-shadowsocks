# IPRanges for Shadowsocks

List all IP ranges from: Google, Bing, Amazon, Microsoft, Azure, Oracle, DigitalOcean, GitHub, Facebook, Twitter, Linode, Yandex, Vkontakte, Telegram, Netflix, Valve with regular auto-updates for Shadowsocks in ACL format .

All lists are obtained from public sources.

## List types

`ipv4.acl`/`ipv6.acl` – list of addresses smallest possible list of CIDRs.

`ipv6.acl` – Temporarily unsupported.
